О проекте

LingDic (лингвистический терминографический словарь-конструктор) представляет собой интерактивное веб-приложение, предназначенное для систематизации и изучения лингвистических аспектов терминологии в сфере искусственного интеллекта путем создния персонализованных словарных статей.

Дидактическая значимость

Данное приложение помогает студентам-лингвистам:

  • Изучать процессы адаптации иностранных терминов ИИ в русском языке
  • Анализировать морфологические, фонетические и семантические особенности заимствований
  • Отслеживать развитие профессионального сленга в сфере ИИ
  • Практиковать навыки терминографического описания новых слов
Технологии

Приложение разработано с использованием следующих технологий:

  • HTML5 и CSS3 для структуры и стилизации
  • JavaScript для интерактивности и управления данными
  • localStorage для сохранения пользовательских словарных статей
  • Адаптивный дизайн для поддержки мобильных устройств
Как добавить новую словарную статью

Для добавления нового термина в словарь:

  1. Нажмите кнопку «+ Добавить слово» в меню слева
  2. Заполните все обязательные поля формы
  3. Используйте панель с буквами с ударением для правильного обозначения ударения в слове
  4. При необходимости добавьте дополнительную информацию
  5. Нажмите кнопку «Добавить слово»

Добавленные термины сохраняются в браузере и будут доступны при следующих посещениях словаря.
