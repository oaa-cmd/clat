Учебный реферат по научному докладу

Учебное веб-приложение для создания рефератов докладов научных конференций по компьютерной лингвистике.

Функциональность приложения

Создание рефератов

Структурированная форма для ввода всех необходимых данных: группа, ФИО, название доклада, ссылка, аннотация и выводы.

Локальное сохранение

Автоматическое сохранение данных в localStorage браузера для предотвращения потери информации.

Экспорт в PDF

Возможность сохранения готового реферата в формате PDF для отправки преподавателю.

Интеграция с ACL

Прямые ссылки на ресурсы Dialog и ACL.

Использованные технологии

Приложение создано с использованием веб-технологий:

  • HTML5 - семантическая разметка
  • CSS3 - стилизация с CSS-переменными и адаптивным дизайном
  • JavaScript ES6+ - интерактивность и работа с данными
  • localStorage API - локальное хранение данных
  • Print CSS - оптимизация для печати/PDF

Задание по курсу "Основы компьютерной лингвистики"

Актуальное состояние компьютерной лингвистики в зеркале Dialog и ACL

Алгоритм выполнения:

  1. Перейдите на сайт конференции по ссылке ниже
  2. Просмотрите материалы конференции
  3. Выберите наиболее интересный для вас доклад
  4. Заполните форму реферата в соответствующем разделе
  5. Сохраните готовый реферат в формате PDF
  6. Отправьте файл преподавателю для проверки
  7. Создайте презентацию выбранного доклада
  8. Выступите с презентацией выбранного доклада
  9. Участвуйте в обсуждении презентаций других докладов

Требования к реферату:

  • Указание группы и фамилии студента
  • Точное название выбранного доклада
  • Прямая ссылка на выбранный доклад
  • Обоснование выбора (что заинтересовало)
  • Краткая структурированная аннотация
  • Личные выводы и размышления
